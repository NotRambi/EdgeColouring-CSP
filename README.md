# EdgeColouring-CSP
CSP in MiniZinc che risolve il problema EdgeColouring

**Come usarlo:**
- Modificare la Matrice di adiacenza nel file GraphToList.py
- Eseguire il comando `python GraphToList.py` o `py GraphToList.py` in base all'OS in cui ti trovi
- Eseguire su MiniZinc il file EdgeColouring.mzn
